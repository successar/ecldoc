INTEGER4 CompareIgnoreCase(STRING src1, STRING src2)
Compares the two strings case insensitively. Returns a negative integer, zero, or a positive integer according to whether the first string is less than, equal to, or greater than the second.
Parameters
src1 The first string to be compared.
src2 The second string to be compared.

UNSIGNED4 Find(STRING src, STRING sought, UNSIGNED4 instance = 1)
