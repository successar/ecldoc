Types.matrix_t DPOTF2 (Types.Triangle tri, Types.dimension_t r, Types.matrix_t A, BOOLEAN clear=TRUE)
Tree
DPOTF2 computes the Cholesky factorization of a real symmetric positive definite matrix A. The factorization has the form A = U**T * U , if UPLO = 'U', or A = L * L**T, if UPLO = 'L', where U is an upper triangular matrix and L is lower triangular. This is the unblocked version of the algorithm, calling Level 2 BLAS.

Parameters
tri indicate whether upper or lower triangle is used
r number of rows/columns in the square matrix
A the square matrix
clear clears the unused triangle
Returns
the triangular matrix requested.

Types.matrix_t DSYRK (Types.Triangle tri, BOOLEAN transposeA, Types.dimension_t N, Types.dimension_t K, Types.value_t alpha, Types.matrix_t A, Types.value_t beta, Types.matrix_t C, BOOLEAN clear=FALSE)
Tree
Implements symmetric rank update C <- alpha A**T A + beta C or c <- alpha A A**T + beta C. C is N x N.

Parameters
tri update upper or lower triangle
transposeA Transpose the A matrix to be NxK
N number of rows
K number of columns in the update matrix or transpose
alpha the alpha scalar
A the update matrix, either NxK or KxN
beta the beta scalar
C the matrix to update
clear clear the triangle that is not updated. BLAS assumes that symmetric matrices have only one of the triangles and this option lets you make that true.
