FUNCTION
Types.matrix_t dpotf2(Types.Triangle tri, Types.dimension_t r, Types.matrix_t A, BOOLEAN clear=TRUE)
DPOTF2 computes the Cholesky factorization of a real symmetric positive definite matrix A. The factorization has the form A = U**T * U , if UPLO = 'U', or A = L * L**T, if UPLO = 'L', where U is an upper triangular matrix and L is lower triangular. This is the unblocked version of the algorithm, calling Level 2 BLAS.
Parameters
tri indicate whether upper or lower triangle is used
r number of rows/columns in the square matrix
A the square matrix
clear clears the unused triangle
Returns
the triangular matrix requested.
