MODULE
BinomialLogisticRegression(UNSIGNED max_iter=200, REAL8 epsilon=Constants.default_epsilon, REAL8 ridge=Constants.default_ridge)
Tree
Binomial logistic regression using iteratively re-weighted least squares.

Parameters
max_iter maximum number of iterations to try
epsilon the minimum change in the Beta value estimate to continue
ridge a value to populate a diagonal matrix that is added to a matrix help assure that the matrix is invertible.
