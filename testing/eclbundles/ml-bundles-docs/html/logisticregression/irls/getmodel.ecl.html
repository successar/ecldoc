FUNCTION
DATASET(Layout_Model) GetModel(DATASET(NumericField) independents, DATASET(DiscreteField) dependents, UNSIGNED max_iter=200, REAL8 epsilon=Constants.default_epsilon, REAL8 ridge=Constants.default_ridge)
Tree
Generate logistic regression model from training data. The size of the inputs is used to determin which work items are processed with purely local operations (the data is moved once as necessary) or with global operations supporting a work item to use multiple nodes.

Parameters
independents the independent values
dependents the dependent values.
max_iter maximum number of iterations to try
epsilon the minimum change in the Beta value estimate to continue
ridge a value to pupulate a diagonal matrix that is added to a matrix help assure that the matrix is invertible.
Returns
coefficient matrix plus model building stats
