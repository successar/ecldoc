DATASET(Layout_Model) GetModel_local(DATASET(NumericField) independents, DATASET(DiscreteField) dependents, UNSIGNED2 max_iter=200, REAL8 epsilon=Constants.default_epsilon, REAL8 ridge=Constants.default_ridge)
Tree
Internal function to determine values for the model co-efficients and selected stats from building the model.

Parameters
independents the independent values
dependents the dependent values.
max_iter maximum number of iterations to try
epsilon the minimum change in the Beta value estimate to continue
ridge a value to populate a diagonal matrix that is added to a matrix help assure that the matrix is invertible.
Returns
coefficient matrix plus model building stats
