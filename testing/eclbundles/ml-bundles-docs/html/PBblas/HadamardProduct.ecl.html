HadamardProduct
DATASET(Layout_Cell) HadamardProduct (DATASET(Layout_Cell) X, DATASET(Layout_Cell) Y)
Element-wise multiplication of X * Y
Element-wise multiplication of X * Y. Supports the "myriad" style interface -- X and Y may contain multiple separate matrixes. Each X will be multiplied by the Y with the same work-item id. Note: This performs element-wise multiplication. For dot-product matrix multiplication, use PBblas.gemm.
Parameters
Y TABLE ( pbblas.layout_cell ) -- A matrix (or multiple matrices) in Layout_Cell form
X TABLE ( pbblas.layout_cell ) -- A matrix (or multiple matrices) in Layout_Cell form
Returns
TABLE ( { UNSIGNED2 wi_id , UNSIGNED4 x , UNSIGNED4 y , REAL8 v } ) A matrix (or multiple matrices) in Layout_Cell form
